# 创建项目

## 创建 Spring Boot 项目

### 脚本

```
使用浏览器打开生成 Spring boot 项目的网站
选择构建工具
设置项目基本信息
设置项目依赖
下载生成的项目
解压后，使用 Eclipse 导入 Maven 项目
这是导入成功后的项目结构
```
